## 🚀 Overview

A high-performance desktop application designed to eliminate digital clutter. This tool scans any directory you give it, analyzes the file extensions, and instantly sorts your mess into a clean, categorized folder structure.

## ⚙️ Features

* **Intuitive GUI:** Built with PyQt5 for a sleek, dark-themed user experience.
* **Smart Categorization:** Automatically creates directories based on file types (.pdf, .jpg, .exe, etc.).
* **Safe-Move Logic:** Intelligent filtering to ensure existing directories and system files aren't disrupted during the organization process.
* **Cross-Platform:** Works wherever Python and PyQt5 are installed.

## 🛠️ Technology Stack

* **UI Framework:** `PyQt5` (Professional-grade GUI library)
* **File System Ops:** `os` & `shutil` (For deep system interaction)
* **Design:** Custom CSS/StyleSheet injection for the "Stark Industries" dark-mode aesthetic.

## 🔧 Installation

1. Install the required framework:
```bash
pip install PyQt5

```


2. Ensure you have the assets:
* Place `file.png` in the same directory as the script for the header image.


3. Run the application:
```bash
python file_organizer.py

```



## 💻 Usage

1. Launch the app.
2. Paste the full path of the "disaster zone" folder (like your Downloads).
3. Hit **Proceed**.
4. Watch the chaos turn into order.
